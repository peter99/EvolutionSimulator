Welcome to EvolutionSimulator

Predicting evolutionary changes since 2014.

What is it?

EvoSim project is meant to predict the genotype of creatures for multiple generations.

Why do I need it?

Not everyone needs this. But if you have some background in biology and want to know what results you should expect, you have come to the right place. And if you are just a random person looking for random code, you have come to the randomly right place. Go on, try this piece of code.

Running the code

Running of this code with custom number of generations to see has not been unabled. The software is under active development currently. Though a large part has been completed, a large part is still left.
